What is CSS

CSS stands for Cascading Style Sheets.

It is a style sheet language used to describe the look and formatting of a document written in HTML (Hypertext Markup Language) or XML (Extensible Markup Language).

CSS separates the presentation of a document from its structure, allowing web designers to control the visual appearance of web pages.

CSS operates on the concept of selectors, which are used to target specific HTML elements and apply styles to them. Selectors can target elements based on their tag name, class, ID, attributes, or their position within the document structure.

How to Add CSS

    Inline CSS: You can add CSS styles directly within the HTML elements using the style attribute.

    Internal CSS: You can include CSS styles within the style tags in the head section of your HTML document.

    External CSS: You can create a separate CSS file with a .css extension and link it to your HTML document using the link tag

CSS Syntax

CSS syntax consists of a set of rules and declarations that define how styles should be applied to HTML elements. The basic structure of CSS syntax is as follows:

selector { property: value; /* Additional properties and values */ }

Let's break down each component of the CSS syntax:

  1. Selector: A selector determines which HTML elements the styles should be applied to. It can be a tag name, class, ID, or other attribute.
    For example, p targets all p elements, .my-class targets elements with the class "my-class," and #my-id targets an element with the ID "my-id."


  2. Property: A property defines a specific aspect of an element's style that you want to change. Examples include color, font-size, background-color, margin, etc.


  3. Value: A value specifies the setting for a particular property. It can be a color, size, length, keyword, or other relevant value. For example, red, 14px, #FFFFFF, italic, etc.


  4. Declaration: A declaration is made up of a property-value pair, enclosed in curly braces {}. It defines how a specific property should be set for the selected element(s). You can have multiple declarations within a selector.


Here's an example that demonstrates the CSS syntax:

h1 { color: blue; font-size: 24px; text-align: center; }

In this example, the selector h1 targets all h1 elements. The subsequent declarations specify that the color should be blue, the font size should be 24 pixels, and the text should be aligned to the center.



CSS syntax allows you to apply styles to multiple selectors by separating them with commas. For example:

h1, h2 { color: green; }

In this case, both h1 and h2 elements will have the color set to green.
