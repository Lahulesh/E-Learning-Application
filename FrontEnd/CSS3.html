What is CSS

CSS stands for Cascading Style Sheets.

It is a stylesheet language used to describe the look and formatting of adocument written in HTML (Hypertext Markup Language) or XML(Extensible Markup Language).

CSS separates the presentation of a document from its structure, allowing web designers to control the visual appearance of web pages.

CSS operates on the concept of selectors, which are used to target specific HTML elements and apply styles to them. Selectors cantarget elements based on their tag name, class, ID, attributes, or their position within the document structure.

How to Add CSS

    Inline CSS: You can add CSS styles directly within the HTML elements using the style attribute.

    Internal CSS: You can include CSS styles within the style tags in the head section of your HTML document.

    External CSS: You can create a separate CSS file with a .css extension and link it to your HTML document using the link tag

CSS Syntax

CSS syntax consists of a set of rules and declarations that define how styles should be applied to HTML elements. The basic structure of CSS syntax is as follows:

selector { property: value; /* Additional properties and values */ }

Let's break down each component of the CSS syntax:

  1. Selector: A selector determines which HTML elements the styles should be applied to. It can be a tag name, class, ID, or other attribute.
    For example, p targets all p elements, .my-class targets elements with the class "my-class," and #my-id targets an element with the ID "my-id."


  2. Property: A property defines a specific aspect of an element's style that you want to change. Examples include color, font-size, background-color, margin, etc.


  3. Value: A value specifies the setting for a particular property. It can be a color, size, length, keyword, or other relevant value. For example, red, 14px, #FFFFFF, italic, etc.


  4. Declaration: A declaration is made up of a property-value pair, enclosed in curly braces {}. It defines how a specific property should be set for the selected element(s). You can have multiple declarations within a selector.


Here's an example that demonstrates the CSS syntax:

h1 { color: blue; font-size: 24px; text-align: center; }

In this example, the selector h1 targets all h1 elements. The subsequent declarations specify that the color should be blue, the font size should be 24 pixels, and the text should be aligned to the center.



CSS syntax allows you to apply styles to multiple selectors by separating them with commas. For example:

h1, h2 { color: green; }

In this case, both h1 and h2 elements will have the color set to green.

CSS Comments

CSS comments are used to add explanatory or descriptive notes within your CSS code. Comments are ignored by the browser and have no impact on the rendering of the web page. They are solely meant for human readability and can be helpful for documentation, organization, or temporarily disabling code snippets.

Single-line comments: To create a single-line comment, you can use the /* */ syntax. Anything between /* and */ is treated as a comment and will not be interpreted by the browser. Here's an example:

/* This is a single-line comment */ p { color: red; }

Multi-line comments: If you need to add comments spanning multiple lines, you can use the /* */ syntax as well. Here's an example:

/* This is a multi-line comment */ p { color: red; }

CSS Colour

CSS provides several ways to define colors, including named colors, hexadecimal colors, RGB values, HSL values, and more. Here are some commonly used methods to specify colors in CSS:

Let's break down each component of the CSS syntax:

  1. Named Colors: CSS has predefined names for a set of colors. For example, you can use red, blue, green, yellow, etc. Here's an example: p { color: red; }

    In this case, the text color of p elements will be set to red.

  2. Hexadecimal Colors: Hexadecimal colors are represented by a hash sign (#) followed by a combination of six hexadecimal digits. Each pair of digits represents the intensity of red, green, and blue (RGB) respectively. For example: p { color: #FF0000; }

    In this case, the text color of p elements will be set to red (#FF0000).

  3. RGB Colors: RGB values define a color using the intensity values of red, green, and blue, each ranging from 0 to 255. The syntax for specifying RGB colors is rgb(red, green, blue). For example: p { color: rgb(255, 0, 0); }

    In this case, the text color of p elements will be set to red.

  4. RGBA Colors: RGBA colors are similar to RGB, but with an additional alpha value to represent opacity. The alpha value ranges from 0 to 1, where 0 is fully transparent, and 1 is fully opaque. Here's an example: p { rgba(255, 0, 0, 0.5); }

    In this case, the text color of p elements to red with 50% opacity.

  5. HSL Colors: HSL (Hue, Saturation, Lightness) values define colors based on their hue, saturation, and lightness. The syntax for specifying HSL colors is hsl(hue, saturation, lightness). For example: p { color: hsl(0, 100%, 50%); }

    In this case, the text color of p elements will be set to a fully saturated red.

  6. HSLA Colors: HSLA colors are similar to HSL, but with an additional alpha value for opacity. The alpha value ranges from 0 to 1. Here's an example: p { color: hsla(0, 100%, 50%, 0.5); }

    In this case, the text color of p elements to a fully saturated red with 50% opacity.

Background Colour

In CSS, the background property is used to define the background styles for an element. It allows you to set various background-related properties such as color, image, position, repeat, and more. Here are some commonly used background-related properties:

Let's break down each component of the CSS syntax:

  1. Background Colors: You can set the background color of an element using the background-color property. For example: body { background-color: #F0F0F0; }

    This code sets the background color of the Body element to a light gray (#F0F0F0).

  2. Background Image: You can set a background image for an element using the background-image property. For example: div { background-image: url("image.jpg"); }

    In this case, the div element will have an image called "image.jpg" as its background.

  3. Background Repeat: The background-repeat property controls how the background image is repeated or positioned. Common values include repeat (default), no-repeat, repeat-x, and repeat-y. For example: div { background-repeat: no-repeat; }

    This code ensures that the background image of the div element is not repeated.

  4. Background Position: The background-position property determines the starting position of the background image. It accepts values such as top, bottom, left, right, center, or specific pixel or percentage values. For example: div { background-position: center; }

    In this case, the background image of the div element will be centered.

  5. Background Size: The background-size property allows you to specify the size of the background image. You can use values like cover, contain, specific pixel dimensions, or percentages. For example: div { background-size: cover; }

    This code ensures that the background image of the div element covers the entire element without distortion.

CSS Border

In CSS, the border property is used to define the style, width, and color of an element's border. It allows you to customize the appearance of the border surrounding an element.

Here are some commonly used border-related properties:

  1. Border Style: The border-style property defines the style of the border. Common values include solid (default), dashed, dotted, double, groove, ridge, inset, and outset. For example: div { border-style: dashed; }

    This code sets the border style of the div element to dashed.

  2. Border Width: The border-width property sets the width of the border. You can specify it using keywords such as thin, medium, or thick, or by specifying a specific pixel value. For example: div { border-width: 2px; }

    This code sets the border width of the div element to 2 pixels.

  3. Border Color: The border-color property sets the color of the border. You can use named colors, hexadecimal colors, RGB values, or other valid color representations. For example: div { border-color: red; }

    This code sets the border color of the div element to red.

  4. Border Radius: The border-radius property is used to create rounded corners for an element's border. It accepts a length value or a percentage value. For example: div { border-radius: 10px; }

    This code creates a border with rounded corners of 10 pixels for the div element.

  5. Shorthand Border Property: You can use the border shorthand property to define all border-related properties at once. The order of the values is width, style, and color. For example: div { border: 1px solid blue; }

    This code sets the border width to 1 pixel, the style to solid, and the color to blue for the div element.

Margin

In CSS, the margin property is used to control the spacing around an element. It defines the empty space between an element's border and the adjacent elements or the outer edges of its containing element. The margin property allows you to specify the spacing on all four sides of an element (top, right, bottom, and left) or set individual margins for each side.

selector { margin-top: value; margin-right: value; margin-bottom: value; margin-left: value; }

Here's an example that sets the margins of a div element:

div { margin-top: 10px; margin-right: 20px; margin-bottom: 10px; margin-left: 20px; }

In this case, the top margin is 10 pixels, the right margin is 20 pixels, the bottom margin is 10 pixels, and the left margin is 20 pixels.



Alternatively, you can use the shorthand margin property to set all margins at once. The order of values is as follows: top, right, bottom, and left. Here's an example:

div { margin: 10px 20px 10px 20px; }

This code achieves the same result as the previous example, setting the top, right, bottom, and left margins to 10 pixels, 20 pixels, 10 pixels, and 20 pixels, respectively.

Padding

In CSS, the padding property is used to control the spacing between the content of an element and its borders. It defines the empty space between an element's content and its border. The padding property allows you to specify the spacing on all four sides of an element (top, right, bottom, and left) or set individual padding values for each side.

selector { padding-top: value; padding-right: value; padding-bottom: value; padding-left: value; }

Here's an example that sets the margins of a div element:

div { padding-top: 10px; padding-right: 20px; padding-bottom: 10px; padding-left: 20px; }

In this case, the top padding is 10 pixels, the right padding is 20 pixels, the bottom padding is 10 pixels, and the left padding is 20 pixels.

Alternatively, you can use the shorthand padding property to set all paddings at once. The order of values is as follows: top, right, bottom, and left. Here's an example:

div { padding: 10px 20px 10px 20px; }

This code achieves the same result as the previous example, setting the top, right, bottom, and left padding to 10 pixels, 20 pixels, 10 pixels, and 20 pixels, respectively.

Box Model

The box model is a fundamental concept in CSS that describes how elements are rendered and how their dimensions and spacing are calculated. It represents the structure of an element as a rectangular box, with various layers that determine its overall size and positioning.



The CSS box model consists of four main components:

  1. Content: The innermost layer of the box model is the actual content of the element, such as text, images, or other HTML elements.
  2. Padding: The padding is the space between the content and the element's border. It provides extra space inside the element to separate its content from the border. The padding can be set using the padding property.
  3. Border: The border is the next layer outside the padding and encloses the content and padding of the element. It can be styled with a specific color, style, and width using the border property.
  4. Margin: The margin is the outermost layer of the box model and provides space between the element and adjacent elements. It creates separation between elements on the page. The margin can be set using the margin property.

    5. The total width and height of an element are calculated by adding the content width/height, padding, border, and margin together. It's important to consider the box model when working with CSS layouts to ensure proper spacing and positioning of elements.

Height

In CSS, the height property is used to set the height of an element. It determines the vertical size of the content box of an element, excluding any padding, border, or margin.

div { height: 200px; }

Width

In CSS, the width property is used to set the width of an element. It determines the horizontal size of the content box of an element, excluding any padding, border, or margin.

div { width: 300px; }

Position

In CSS, the position property is used to control the positioning of elements on a web page. There are several values that can be used with the position property, each with its own behavior. Here are the commonly used values with examples:

  1. Static: This is the default value, and elements with position: static are positioned according to the normal flow of the document. They are not affected by the top, bottom, left, right, or z-index properties. .static-example { position: static; }
  2. Relative: relative, an element is positioned relative to its normal position. You can use the top, bottom, left, and right properties to move the element from its original position. .relative-example { position: relative; top: 20px; left: 30px; }
  3. Absolute: When an element has position: absolute, it is positioned relative to its closest positioned ancestor (or the document if there is no positioned ancestor). This value allows you to precisely position an element on the page. .absolute-example { position: absolute; top: 50px; left: 100px; }
  4. Fixed: Elements with position: fixed are positioned relative to the viewport, meaning they remain fixed even when the page is scrolled. This is commonly used for elements like navigation bars or sticky headers. .fixed-example { position: fixed; top: 0; right: 0; }
  5. Sticky: With position: sticky, an element is positioned based on the user's scroll position. It acts as relative within its parent until a specified threshold is reached, and then becomes fixed. It's often used for elements that should "stick" to a certain position while scrolling. .sticky-example { position: sticky; top: 20px; }

These are the main values of the position property in CSS. By using these values and adjusting the corresponding properties (top, bottom, left, right), you can control the positioning of elements on your web page.

Fonts

CSS fonts refer to the CSS properties and rules used to specify and style the typography and fonts of text within a webpage. CSS provides various properties to control the appearance, size, family, weight, and other aspects of fonts.

  1. Font-family: The font-family property specifies the font family or a list of font families for text. It allows you to define the preferred font for an element. For example: p { font-family: Arial, sans-serif; }

    In this case, the P element's text will be displayed using the Arial font if available, and if not, a generic sans-serif font will be used as a fallback.

  2. Font-size: The font-size property determines the size of the font. It can be specified using different units, such as pixels (px), percentages (%), ems (em), or other valid CSS length units. For example: p { font-size: 16px; }

    This code sets the font size of the P element to 16 pixels.

  3. Font-weight: The font-weight property sets the thickness or boldness of the font. It accepts values such as normal, bold, bolder, lighter, or numeric values ranging from 100 to 900. For example: p { font-weight: bold; }

    In this case, the text in the P element will be displayed in a bold font weight.

  4. Font-style: The font-style property specifies the style of the font, such as normal or italic. For example: p { font-style: italic; }

    This code sets the text in the P element to be displayed in italic style.

  5. Text-decoration: The text-decoration property controls text decoration, such as underlining, overlining, line-through, or none. For example: p { text-decoration: underline; }

    In this case, the text in the P element will be underlined.

Icons

In CSS (Cascading Style Sheets), icons refer to small, graphical representations used to convey meaning or represent actions or objects on a website or web application. These icons are typically created as vector graphics, allowing them to scale without losing quality. They are widely used in user interfaces (UI) to improve user experience and provide visual cues for various elements

  1. Navigation: Icons are often used to represent links or menu items, making it easier for users to identify different sections of a website.
  2. Actions: Icons can represent common actions like "home," "search," "settings," "add," "delete," "edit," etc.
  3. Status Indicators: Icons can be used to indicate the status of a process, such as "loading," "success," "error," or "warning."
  4. Social Media: Social media icons are commonly used to link to a website's social media profiles.
  5. File Types: Icons can indicate the type of a file, such as PDF, image, video, etc.

    CSS provides several ways to include icons in a web page. Some common methods include:

  1. Font Icons: Using special font files that contain icon glyphs, such as Font Awesome, Material Icons, or Ionicons. You can display icons by adding the appropriate class to an HTML element and setting the font-family to the icon font.
  2. SVG Icons: Including SVG (Scalable Vector Graphics) files directly in the HTML or CSS and using them as background images or inline elements.
  3. Icon Libraries: Utilizing CSS libraries or frameworks like Font Awesome or Bootstrap, which provide a set of pre-designed icons that you can easily use in your project.

Outline

The outline property in CSS is used to create a visible outline around an element, typically used for highlighting or indicating focus on interactive elements like links, form fields, or buttons. It is similar to the border property but is different in how it interacts with the element's box model.

The basic syntax of the outline property is as follows:

outline: [outline-width] [outline-style] [outline-color];
  1. outline-width: This specifies the thickness of the outline. It can have values like thin, medium, thick, or a specific length (e.g., 2px).
  2. outline-style: This determines the style of the outline. Common values are dotted, dashed, solid, double, groove, ridge, inset, outset, or none.
  3. outline-color: This sets the color of the outline. It can take any valid CSS color value (e.g., red, #00ff00, rgba(255, 0, 0, 0.5)).
    4. button { outline: 2px dashed blue; }

In this example, when you focus on a button element, it will have a dashed blue outline with a thickness of 2 pixels.

It's important to note that the outline property does not take up space in the layout of the page. Therefore, if you use it to highlight an element, it won't affect the positioning or dimensions of that element.

Display

The display property can take various values, each affecting the element's layout differently.
Here are some common values for the display property:

  1. block: The element generates a block-level box, taking up the full width available and starting on a new line. Block-level elements have a line break before and after them. Examples of block-level elements are Div, P, H1 to H6, and UL
  2. inline: The element generates an inline-level box, taking up only as much width as necessary for its content. Inline-level elements do not cause line breaks before or after them. Examples of inline elements are span, a , strong, and em.
  3. inline-block: This value combines aspects of both block and inline. The element generates an inline-level box, but you can set a specific width and height for it. It does not cause line breaks before or after, and multiple inline-block elements can appear on the same line. This is often used to create elements that flow horizontally like inline elements but can have a width and height like block elements.
  4. flex: The element becomes a flex container, and its children become flex items. It allows you to use the Flexbox layout model for arranging elements within the container along a flex axis (either horizontally or vertically).
  5. grid: The element becomes a grid container, and its children become grid items. It allows you to use the CSS Grid layout model for arranging elements in rows and columns within the container.
  6. none: The element is not displayed at all. It will be removed from the flow of the document as if it doesn't exist, and its space will not be taken up by other elements.
  7. inline-flex: This is similar to flex, but the container behaves as an inline-level element with flex layout.
  8. table, table-cell, table-row: -These values allow you to create table-like structures without using actual HTML table elements. They are part of the CSS table layout model.

    9. The display property is a powerful tool for controlling the layout of elements and is often used in combination with other CSS properties to create responsive and visually appealing designs.

Z-Index

In CSS, the z-index property is used to control the stacking order of positioned elements on a web page. It determines the vertical position of an element along the z-axis, which is the axis perpendicular to the screen. Elements with a higher z-index value will appear in front of elements with a lower value.

The z-index property only works on elements that have a position value other than static. The most commonly used values for position are relative, absolute, fixed, and sticky.

element { z-index: value; }

The value can be either a positive integer, zero, or a negative integer:

  1. Positive integer: The element is stacked above elements with lower z-index values and below elements with higher values.
  2. Zero: The element is stacked in its default order. Elements with z-index: 0 will be placed below elements with positive z-index values.
  3. Negative integer: The element is stacked below elements with positive z-index values and above elements with lower values.

OverFlow

In CSS3, the overflow property is used to control how content that overflows the bounding box of an element is displayed. It defines how the content is handled when it exceeds the dimensions of the container or if it is larger than the space available to display it.

    visible: Any content that overflows the container will be clipped and not visible. It will be effectively hidden, and users won't be able to see it.

    hidden: You can include CSS styles within the style tags in the head section of your HTML document.

    scroll: The container will display horizontal and vertical scrollbars, allowing users to scroll and see the entire content that overflows the container.

Float

In CSS, "float" is a property that allows an element to be positioned along the left or right side of its containing element, while allowing other content to flow around it. When an element is floated, it is taken out of the normal document flow, which means other block-level elements will ignore the space it occupies, and they will wrap around it instead.

element { float: left | right | none | initial | inherit; }
  1. left: The element is floated to the left side, and content will wrap around its right side.
  2. right: The element is floated to the right side, and content will wrap around its left side.
  3. none (default): The element is not floated, and it will be placed in the normal document flow.
  4. initial: The property takes the default value.
  5. inherit: The property takes the value from its parent element.

Opacity

In CSS, the opacity property is used to control the transparency or the level of opacity of an element, including all of its child elements. It allows you to make an element partially transparent, revealing the content behind it. The opacity property accepts values between 0 and 1, where 0 represents completely transparent (invisible), and 1 represents completely opaque (fully visible).

element { opacity: value; }
  1. value: A numerical value between 0 and 1. The value 0 makes the element completely transparent, and the value 1 makes it fully opaque.
    2. Example: .transparent-element { opacity: 0.5; /* The element and its content will be 50% transparent */ }

It's important to note that the opacity property affects the transparency of an element and all its child elements. This means that if you set an element to be partially transparent, all the content within that element will also have the same transparency level.
